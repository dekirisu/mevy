# API Reference

Complete reference documentation for mevy's macro features.

## Overview

| Reference | Description |
|---|---|
| [CSS-like Fields & Slim Aliases](ui-fields.md) | All `ui!{}` fields with their slim shortcuts |
| [Entity Selectors](entity-selectors.md) | `entity!{}` selector syntax |
| [Grid Track Syntax](grid-tracks.md) | Grid track sizing functions |

## Quick Navigation

- **`ui!{}` fields**: Position, visibility, transform, size, margin, padding, box styling, text styling, images, flex, grid, overflow, and more — with slim aliases in the same table.
- **`entity!{}` selectors**: World types (`<world>`, `<+world>`, etc.), entity queries (`<|#Marker>`), resource selectors (`<|@Resource>`), and redirection chains.
- **Grid tracks**: `1px`, `3%`, `10fr`, `min_content`, `max_content`, `minmax(...)`, `repeat(...)`, `auto_fill`, `auto_fit`, `fit_content`.
